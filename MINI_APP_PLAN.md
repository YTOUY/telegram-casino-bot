## 🎯 Типы кнопок и их стили:

### 1. **Основные кнопки (Primary)**
   - Большие, яркие, привлекающие внимание
   - Используются для главных действий (Игры, Кошелек, Профиль)

### 2. **Вторичные кнопки (Secondary)**
   - Среднего размера
   - Используются для дополнительных действий

### 3. **Кнопки ставок (Betting)**
   - Компактные, с коэффициентами
   - Группируются по типам ставок

### 4. **Кнопки навигации (Navigation)**
   - Кнопки "Назад", "Дальше", "Назад к..."
   - Меньшего размера, внизу страниц

### 5. **Кнопки действий (Action)**
   - Кнопки подтверждения, отмены
   - Выделяются цветом (зеленый/красный)

### 6. **Кнопки сумм (Amount)**
   - Быстрый выбор суммы ($10, $20, и т.д.)
   - Компактные, в сетке

## 🎨 Стили и анимации:

### Цветовая схема:
- **Основной цвет**: Темно-синий/Фиолетовый (#1a1a2e, #16213e)
- **Акцентный цвет**: Зеленый для выигрышей (#00ff88)
- **Цвет проигрыша**: Красный (#ff4444)
- **Нейтральный**: Серый (#888888)

### Анимации:
- **Hover эффект**: Подсветка при наведении
- **Click эффект**: Анимация нажатия (scale)
- **Загрузка**: Skeleton loader для данных
- **Уведомления**: Всплывающие уведомления (toast)
- **Переходы**: Плавные переходы между страницами
- **Игры**: Анимация броска кубика/дартса и т.д.

### Типографика:
- **Заголовки**: Жирный, крупный шрифт
- **Текст**: Средний размер, читаемый
- **Цифры**: Моноширинный шрифт для балансов и сумм
- **Эмодзи**: Используются для визуального разделения
