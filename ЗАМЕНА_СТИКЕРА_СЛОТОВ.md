# Как заменить стикер для слотов через команду /sticker

## Процесс замены стикера слотов:

1. **Отправьте команду `/sticker` боту** (только для администраторов)

2. **Отправьте новый стикер для слотов** (один стикер)

3. **Отправьте сообщение с названием:**
   ```
   1 slots_base
   ```

4. **Готово!** Стикер будет сохранен в базу данных и будет использоваться в мини-приложении.

## Примечания:

- Название стикера для слотов должно быть **`slots_base`** (это базовый стикер, который отображается на кнопке игры)
- Если стикер с таким названием уже существует, он будет **обновлен** новым стикером
- Для отмены отправьте `/cancel`
- После замены стикер будет автоматически использоваться в мини-приложении через API

## Пример использования:

1. Отправьте: `/sticker`
2. Отправьте стикер (например, новый анимированный стикер для слотов)
3. Отправьте: `1 slots_base`
4. Получите подтверждение: `✅ Сохранено стикеров: 1`

---

**Важно:** Стикер будет загружаться через API сервера, поэтому убедитесь, что бот работает и имеет доступ к базе данных.
